# Admin Module

هذا المجلد يحتوي على جميع مكونات الإدارة في الموقع.

## الهيكل

```
src/admin/
├── components/          # مكونات الإدارة
│   ├── AdminCard.tsx   # بطاقة الصفحة
│   ├── AdminButtons.tsx # أزرار التحكم
│   └── index.ts        # تصدير المكونات
├── styles/             # ملفات التصميم
│   └── AdminStyles.ts  # الستايلات المشتركة
├── utils/              # الأدوات المساعدة
│   └── adminUtils.ts   # دوال الإدارة
├── AdminPanel.tsx      # لوحة الإدارة الرئيسية
├── ContentEditor.tsx   # محرر المحتوى
├── QuickLogin.tsx      # تسجيل دخول سريع
├── index.ts           # تصدير المكونات الرئيسية
└── README.md          # هذا الملف
```

## المكونات

### AdminPanel
لوحة الإدارة الرئيسية التي تعرض جميع الصفحات القابلة للتعديل.

### ContentEditor
محرر المحتوى لتحرير محتوى الصفحات المختلفة.

### QuickLogin
صفحة تسجيل الدخول السريع للمستخدمين المصرح لهم.

## الاستخدام

```typescript
import { AdminPanel, ContentEditor, QuickLogin } from '../admin';
```

## الميزات

- ✅ إدارة محتوى جميع الصفحات
- ✅ واجهة مستخدم جميلة ومتجاوبة
- ✅ تسجيل دخول سريع للمستخدمين المصرح لهم
- ✅ تحرير محتوى الصفحات بسهولة
- ✅ تصميم منظم وقابل للصيانة
